# Nikhil Tathe — Portfolio Website

A modern, single-page portfolio website showcasing my professional experience, skills, education, awards, and contact information.

## Live Site

Enable **GitHub Pages** in your repo settings (Settings → Pages → Source: `main` branch, root `/`) to host at:  
**https://nikhiltathe.github.io/AboutMe/**

## Features

- **Responsive layout** with fixed top bar and left sidebar navigation
- **Interactive work experience timeline** spanning 15+ years across 4 companies
- **Skill tags** organized by category with hover effects
- **Education cards** with CGPA/percentage badges
- **Awards & certificates** section with year badges
- **Contact section** with email, phone, and LinkedIn links
- **Smooth scroll** navigation and scroll-triggered fade-in animations
- **Mobile-friendly** with hamburger menu and collapsible sidebar
- **Resume download** button linking directly to the PDF in the `Resume/` folder

## Resume Updates

Update the PDF file in the `Resume/` folder and push to `main`. The download button on the website will automatically serve the latest version.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Inter](https://fonts.google.com/specimen/Inter) font via Google Fonts
- [Font Awesome 6](https://fontawesome.com/) icons via CDN
